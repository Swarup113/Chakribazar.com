# Online Job Portal (Chakribazar.com)

## Overview
Chakribazar.com is a web-based solution designed to bridge the gap between job seekers and employers. It aims to replace the time-consuming manual hiring process with an efficient, automated system. The platform allows organizations to post vacancies and manage applications, while candidates can easily browse jobs, submit resumes, and apply for positions—all in one place.

## Key Features
- Dual User Roles: Distinct interfaces for Job Seekers and Employers.
- Job Seeker Dashboard:
- Register, Login, and Profile Management.
- Search for jobs by category.
- Apply for multiple jobs and upload multiple resumes.
- Prevention of duplicate applications for the same job.
- Employer Dashboard:
- Post new job vacancies and delete expired posts.
- View applicant profiles and download resumes.
- Accept or reject applications directly through the dashboard.
- Search candidates based on qualifications.

## Tech Stack
- Frontend: HTML, CSS
- Backend: PHP
- Database: MySQL
- Server Environment: XAMPP
